Our team of four MS-HCI students has designed and prototyped the Great Circles Smart Key system, a sequence of interactions involving a tangible key object and an interface projected on a car window that aims to help users engage in efficient ride share activity.

The system aims to immerse interactors in an environment through the tangible key object to perform tasks like reserving a car, performing scheduling activity or going for a quick ride. This is useful to respect our user-centric goals and complete tasks in a few clicks. This interface will replace having to navigate through a traditional mouse and keyboard interface for most use cases, while providing the alternate interface when the user requires it.
